USE O MAILCHIMP FORM:

Para usar o formulário mailchimp, você precisa de um URL mailchimp. Para obter o URL mailchimp, faça o login para sua conta mailchimp, clique no menu de lista, clique no menu de download de estatísticas e selecione formulários de inscrição.
Em seguida, escolha formulários incorporados e selecione o formato nu. No código do formulário gerado, obtenha o valor do atributo ACTION do formulário e use-o como seu URL de mailchimp. Abra o arquivo main.js localizado
na pasta "js" do modelo e procure por esta linha de código:

var cfg = {
    scrollDuration : 800, // smoothscroll duration
    mailChimpURL   : 'https://facebook.us8.list-manage.com/subscribe/post?u=cdb7b577e41181934ed6a6a44&amp;id=e6957d85dc'   // mailchimp url
},

Substitua o valor de mailChimpURL pelo seu URL mailchimp.
